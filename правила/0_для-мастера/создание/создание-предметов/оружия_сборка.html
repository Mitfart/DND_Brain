Оружия_Сборка

Правила:

  1. У каждого Оружия есть своё количество Очков, за которые можно закупать свойства при его крафте (стартовое Простое - 0, Воинское - 1)
  2. Количество Очков зависит от качества материала и типа оружия
  3. Примерная цена рассчитывается по формуле, но может определятся мастером в зависимости от Сюжета:
    • 10 + 25 x (1 + КО ^ 2) x КС
      • КО - Количество Очков
      • КС - Количество Свойств
